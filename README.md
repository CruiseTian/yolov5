<p>yolo格式的标签为txt格式的文件，文件名跟对应的图片名一样，除了后缀改为了.txt。
具体格式如下：</p>
<ul>
<li>每个目标一行，整个图片没有目标的话不需要有txt文件</li>
<li>每行的格式为<code>class_num x_center y_center width height</code></li>
<li>其中<code>class_num</code>取值为<code>0</code>至<code>total_class - 1</code>，框的四个值<code>x_center</code> <code>y_center</code> <code>width</code> <code>height</code>是相对于图片分辨率大小正则化的<code>0-1</code>之间的数，左上角为<code>(0,0)</code>，右下角为<code>(1,1)</code></li>
</ul>

<li><code>--bucket</code>使用gsutil bucket</li>
<li><code>--cache-images</code> 使用缓存图片训练</li>
<li><code>--image-weights</code> 训练中对图片加权重</li>
<li><code>--multi-scale</code> 训练图片大小+/-50%变换</li>
<li><code>--single-cls</code> 单类训练</li>
<li><code>--adam</code> 使用torch.optim.Adam()优化器</li>

<li><code>--log-artifacts</code> 输出artifacts,即模型效果</li>

<p>有参：</p>
